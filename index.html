The world Is My Playground

HTML or otherwise known as (hyper text markup language) is the structure of your website. I will be using html to layout the basic side of your site such as texts and sections that i create using dividers which i will then go and edit within CSS. My HTML code is done by the specifications set by W3.Org.

CSS or otherwise known as (Cascading Style Sheets) is the language i will be using to bring your site visualy top life. This is where all the images and beauty get applied to the site. I will do this using multiple techniques that i have learned from multiple sources. It will be very clean code so that anyone else may edit it with no issues of finding what code is controlling certian eliments.

I am currently in the process of learning Javascript. It is still a new language to me but i am devoting alot of time and energy to understanding it and applying it to not only this site. But to any sites i have the luxury of developing in the future of my career. Once i have finished learning the basics of the Javascript language i will be updating this page with the latest standards put out by w3.org as well.
